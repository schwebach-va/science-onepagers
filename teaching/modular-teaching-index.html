A Modular Index: Finding the Right Piece for Whatever You're Already Teaching

DRAFT — review before publishing

Everybody's Science says this about itself, in the note before Chapter One: “Most teachers won't use this book cover to cover as a single course text, and it isn't built assuming you will — pull whichever chapter matches whatever unit you're already teaching, in whatever order your course actually moves in.” This page is that promise made usable. It is not a syllabus and it does not imply a front-to-back teaching sequence — it's a lookup tool. Find the chapter or one-pager that matches whatever you're already covering this week, and go straight there.

Type below to filter by chapter title, topic, or one-pager name. The table itself holds the full map: every chapter in the book, the existing one-pager(s) already built to pair with it, the closest matching NGSS codes, and — where one is planned — the Teaching-tab companion piece that goes with it.
